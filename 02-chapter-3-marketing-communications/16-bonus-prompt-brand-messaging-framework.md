## Bonus Prompt: Brand Messaging Framework

As a brand strategist, create a comprehensive messaging framework for our [PRODUCT/SERVICE] that will guide all marketing communications.





Include the following elements:

- Brand essence (core purpose in one sentence)
- Key audience segments with primary pain points for each
- Central value proposition with supporting proof points
- Brand voice characteristics with do's and don'ts
- Message hierarchy for different communication channels
- 
- Competitor messaging analysis highlighting differentiation opportunities




Format this as a strategic brand messaging guide that ensures consistency across all touchpoints.
